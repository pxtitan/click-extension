# click-extension

A minimal Chrome extension that searches whatever text is selected on screen by pressing **CTRL + ENTER**. It opens a new tab with Google search results for the selected text.

## Usage

1. Load the extension in Chrome via `chrome://extensions` → **Load unpacked** → select this folder.
2. Select any text on any webpage.
3. Press **CTRL + ENTER** — a new tab opens with the Google search results.
